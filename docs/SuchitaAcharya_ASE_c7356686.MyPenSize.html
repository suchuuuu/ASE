Remarks

The MyPenSize class is responsible for validating and applying the pen size. It ensures the pen size:

  • Is a non-negative integer.
  • Does not exceed 100.
If validation fails, a BOOSE.RestrictionException is thrown.

Execute()

Executes the PenSize command to set the pen size on the Canvas.

public override void Execute()

Examples

The following example demonstrates how to use the MyPenSize class:

Canvas myCanvas = new Canvas();
PenSize penSizeCommand = new PenSize(myCanvas, 10);
penSizeCommand.Execute();

Remarks

The method performs the following steps:

  1. Extracts the pen size parameter from the base class.
  2. Validates that the pen size is within the valid range (0 to 100).
  3. Sets the pen size using the SetPenSize(int) method.
If the pen size is invalid (either too large or negative), a BOOSE.RestrictionException is thrown.

Exceptions

RestrictionException

Thrown when the pen size exceeds 100 or is negative.
